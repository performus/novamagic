# Стартовый промпт для Claude Code — Фаза 0 (каркас)

> Скопируй текст ниже в Claude Code. Положи рядом `SPEC.md` в корень проекта, чтобы он мог на неё ссылаться.
> Цель Фазы 0: пустой каркас из 11 блоков на localhost. Без вёрстки блоков, без анимаций.

---

Мы строим премиальный лендинг **NOVAMAGIC — Dubai Wind** (парфюмированный концентрат для стирки). Полная спецификация — в файле `SPEC.md` в корне, читай её как источник истины. Сейчас выполняем **только Фазу 0 (§8)** — каркас. Не переходи к вёрстке блоков и анимациям.

**Стек (зафиксирован):** Vite + React + TypeScript, стили — CSS Modules + CSS-переменные. GSAP/Lenis **не ставим** на этой фазе.

**Задачи Фазы 0:**

1. Инициализируй проект: `npm create vite@latest novamagic -- --template react-ts`, установи зависимости.

2. Подключи шрифт **Work Sans** (Google Fonts, веса 400/500/600) — через `<link>` в `index.html` или `@fontsource/work-sans`.

3. Создай структуру папок по §3 спеки:
   ```
   public/{renders,notes,sequences/hero-bottle}
   src/styles/{tokens.css,reset.css,globals.css}
   src/components/{ui,layout}
   src/sections/{Hero,Intro,WarmScent,FragrancePyramid,ThreeInOne,Features,Ritual,Composition,Closing}
   src/animation/        (пустая, для Фазы 2)
   ```

4. Создай `src/styles/tokens.css` со **всеми** токенами из §4 спеки (цвета, типографика, стекло, радиусы, spacing). Подключи tokens + reset + globals в `main.tsx`. Установи фон страницы `--bg-cream`, базовый шрифт `--font`.

5. Создай переиспользуемый компонент-заглушку `SectionPlaceholder` (props: `id`, `label`, `minHeight`). Рендерит блок заданной высоты с подписью по центру и тонкой рамкой `--line`, чтобы границы блоков были видны.

6. В `App.tsx` отрендери **11 блоков-заглушек** в порядке макета, с примерными высотами (пропорции из фрейма 1920×7911):
   | id | label | minHeight |
   |---|---|---|
   | header | B0 — Header / Nav | 90px (sticky сверху) |
   | hero | B1 — Hero | 1089px |
   | intro | B2 — Intro (glass) | 490px |
   | warm-scent | B3 — Warm Scent | 240px |
   | pyramid | B4 — Пирамида: Начало/Сердце/Финал | 1100px |
   | three-in-one | B5 — 3-в-1 | 400px |
   | features | B6 — Features grid | 2000px |
   | ritual | B7 — Ritual | 200px |
   | composition | B8 — Состав/характеристики | 850px |
   | closing | B9 — Дорогое звучание | 600px |
   | footer | B10 — Footer | 520px |

7. Базовый контейнер: контент по центру, `max-width: var(--container-max)`, поля по краям. Каждый блок — на всю ширину, контент внутри контейнера.

8. Запусти `npm run dev`. Убедись, что на localhost видна вертикальная лента из 11 подписанных блоков на кремовом фоне, без ошибок в консоли.

**Важно:**
- Реальную вёрстку, копирайт-текст, картинки и анимации **не добавляй** — это Фаза 1 и 2.
- Текст в блоках будем выносить в `content.ts` на блок (i18n-дружелюбно) — но это на Фазе 1.
- После запуска **остановись** и покажи мне результат на localhost. Дальше пойдём блок за блоком (Фаза 1), по одному, с проверкой после каждого.

Когда закончишь Фазу 0 — дай команду запуска и краткий чек-лист, что готово.
